# MusicPlayer v0.1.0 项目脚手架 · 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 搭建 Tauri 2.x + React 19 + TypeScript 最小可运行项目骨架，包含前端路由壳、Rust 模块骨架、SQLite V1 迁移，能在 `pnpm tauri dev` 下启动并看到 React 页面。

**Architecture:** Tauri 2.x 单进程双语言架构 — WebView (React 19 + Vite + Tailwind v4) 通过 Tauri IPC 与 Rust 核心通信。前端用 Zustand 管理 UI 状态，Rust 侧按 commands/player/library/metadata/db/system 分模块。

**Tech Stack:** Tauri 2.10.x, React 19.2, TypeScript strict, Vite, Tailwind CSS 4.2, pnpm, Rust 1.95, rusqlite (bundled), symphonia, cpal, lofty, notify

**当前项目状态：** 仅有 CLAUDE.md + 设计文档 + .gitignore + Git init，零代码。

```rust
use serde::Serialize;

/// 统一错误类型，经 Tauri IPC 传到前端
#[derive(Debug, thiserror::Error, Serialize)]
#[serde(tag = "kind", content = "message")]
pub enum AppError {
    #[error("数据库错误: {0}")]
    Database(#[from] String),

    #[error("文件未找到: {0}")]
    FileNotFound(String),

    #[error("无权限访问: {0}")]
    PermissionDenied(String),

    #[error("播放失败: {0}")]
    Playback(String),

    #[error("元数据解析失败: {0}")]
    Metadata(String),

    #[error("扫描错误: {0}")]
    Scan(String),

    #[error("无效输入: {0}")]
    InvalidInput(String),

    #[error("内部错误: {0}")]
    Internal(String),
}

// Tauri 要求错误实现 serde::Serialize
impl From<rusqlite::Error> for AppError {
    fn from(e: rusqlite::Error) -> Self {
        AppError::Database(e.to_string())
    }
}

pub type AppResult<T> = Result<T, AppError>;
```

## 风险与回退

| 风险 | 缓解 |
|------|------|
| symphonia 对部分 FLAC/ALAC 变体解码失败 | Phase 3 spike 阶段验证，必要时 fallback ffmpeg-rs |
| cpal 在特定 macOS 音频设备上行为异常 | 备选：用 Tauri 侧 AVAudioEngine 桥接 |
| Tauri 2.x macOS 沙盒 bookmark 权限过期 | 每次扫描前重新请求权限 |
| Tailwind v4 与 Radix UI 样式冲突 | Radix 组件提供 className 透传，用 Tailwind 覆盖 |
